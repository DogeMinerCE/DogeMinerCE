### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/DogeMinerCE/DogeMinerCE.git
   cd DogeMinerCE
   ```

2. Open `play/index.html` in your web browser
3. Start mining Dogecoins!
